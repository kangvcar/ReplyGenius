ReplyGenius Privacy Policy

TL;DR: We don't collect, store, or share your personal data. Everything stays on your device.

1. Information We Collect

Data We Store Locally

  • Configuration Settings: Your API keys, preferred reply styles, language settings, and custom styles
  • Extension Preferences: UI settings and user preferences

Data We Process Temporarily

  • Tweet Content: Only the text of tweets you choose to reply to, processed momentarily for AI generation

Data We Don't Collect

  • Personal information or user profiles
  • Browsing history outside Twitter/X
  • Generated replies or conversation history
  • Social media account information
  • Location data
  • Device information beyond basic Chrome extension APIs

2. How We Use Your Information

  • Core Functionality: To generate AI-powered replies based on your selected style and preferences
  • Configuration Storage: To save your settings locally for consistent experience
  • API Communication: To send tweet content to your configured AI service for reply generation

3. Data Storage and Security

Local Storage

  • All settings stored locally using Chrome's secure storage API
  • API keys encrypted and stored only on your device
  • No data transmitted to our servers
  • You have complete control over your data

Security Measures

  • HTTPS-only API communications
  • Input sanitization to prevent injection attacks
  • Rate limiting to prevent abuse
  • No persistent storage of sensitive data

4. Third-Party Services

AI Service Providers

  • Purpose: Used only when you trigger reply generation
  • Data Sent: Only the tweet text you're replying to
  • Control: You configure which AI service to use
  • No Persistent Storage: No data stored by ReplyGenius with third parties

Supported AI Services: OpenAI, Anthropic Claude, and other OpenAI-compatible APIs. Each service has its own privacy policy that governs how they handle the data you send.

5. Data Sharing

We do not share, sell, or distribute your data to any third parties except:

  • AI Services: Only tweet text sent to your configured AI provider for reply generation
  • Legal Requirements: If required by law (though we have no data to share)

6. Your Rights and Control

  • Complete Control: All data stored locally, accessible only to you
  • Delete Data: Use "Reset Configuration" to delete all stored data
  • Modify Settings: Change or remove any stored preferences at any time
  • Uninstall: Removing the extension deletes all local data
  • No Account Required: Fully anonymous usage

7. Data Retention

  • Local Settings: Stored until you delete them or uninstall the extension
  • Tweet Content: Not stored; processed in memory only during reply generation
  • Generated Replies: Not stored by ReplyGenius

8. Children's Privacy

ReplyGenius does not knowingly collect personal information from children under 13. Since we don't collect personal information at all, the extension can be safely used by users of all ages.

9. International Users

ReplyGenius works entirely locally on your device. No data crosses borders unless you configure an AI service located in another country.

10. Changes to Privacy Policy

We may update this privacy policy occasionally. Changes will be posted on this page with an updated "Last Modified" date. Significant changes will be communicated through the extension update notes.
